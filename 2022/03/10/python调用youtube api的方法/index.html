创建api key

api key千万不要泄露给他人，否则别人会使用你的身份去做某些事情。如果发现泄露了，请立即删除重新创建。

  1. 进入谷歌api网址
  2. 新建项目
  3. 搜索并启用“YouTube Data API v3”
  4. 创建凭据
  5. 选择“YouTube Data API v3”
  6. 选择公开数据
  7. 复制api密匙

谷歌api的GitHub里说明了Windows该如何进行安装，在cmd中pip进行安装前，需要开启cmd代理，否则会失败。
